Use the following steps to download the FootNote Google Chrome extension (download)

Step 1: Download the file, extract it, then open up Google Chrome.

Step 2: Select the three dots icon in the top right corner of your browser, then go to more tools > extensions.

Step 3: Enable developer mode in the top right hand corner, then select Load Unpacked on the left side.

Step 4: Select the unzipped version of the downloaded folder.

Step 5: You should now see an f icon where chrome extension appear. Select the F icon in the top right hand corner of the Chrome page, then follow the registration process. (To reduce errors and bricking your extension, type in exact correct values. If you break your extension or it crashed, just re-download it.)

Step 6: Re-enter your username and password when asked. Now youn can start commenting on web pages and see other user's comments!

Special Notes PLEASE READ:

If your verification code doesn't come through to your email or you don't get the code but you're on the verification page, just wait two minutes. There is a timeout functionaily that will remove your registration request after two minutes.

PLEASE READ: Also, if the buttons aren't responding, try clicking away from the popup and clicking back onto the extension. It can be a little buggy in the registration page, but once you're logged in everything works pretty smoothly.
