Use the following steps to download the FootNote Google Chrome extension.

Step 1: Go on your browser and search https://people.rit.edu/ajl3545/FootNote/

Step 2: Download the file, extract it, then open up Google Chrome.

Step 3: Select the three dots icon in the top right corner, then go to more tools > extensions.

Step 4: Enable developer mode in the top right hand corner, then select Load Unpacked.

Step 5: Select the unzipped version of the downloaded file.

Step 6: Select the F icon in the top right hand corner of the Chrome page, then follow the registration process.

Step 7: Re-enter your username and password when asked. Now youn can start commenting on web pages and see other user's comments!
